# NEEDSBOXES Theme

This is the official theme for the NEEDSBOXES store, a customized version of the Raed theme, fully aligned with the NEEDSBOXES brand identity.

## Overview

This theme is designed to provide a modern and seamless shopping experience for customers of the NEEDSBOXES store. It is built on top of the Salla Twilight theme framework and uses Tailwind CSS for styling.

## Brand Identity

The theme is designed to reflect the NEEDSBOXES brand identity, as defined in the `deepseek.html` file. This includes:

-   **Colors**: A specific color palette with primary, accent, and neutral colors.
-   **Typography**: The Tajawal and Almarai fonts for Arabic, and Inter for English.
-   **Spacing & Radius**: A consistent spacing and border-radius system.
-   **Shadows**: Soft ambient drop shadows on cards and modals.
-   **Imagery**: Custom imagery and icons that match the brand.

## Getting Started

To use this theme, you need to have a Salla store and the Salla CLI installed.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/bakhe8/needboxes.git
    ```

2.  **Install the dependencies:**
    ```bash
    npm install
    ```

3.  **Run the theme preview:**
    ```bash
    salla theme preview
    ```

## Main Features

The theme includes a variety of features and components to enhance the shopping experience, including:

-   A custom header and footer.
-   A responsive design that works on all devices.
-   A variety of home page components to showcase products and promotions.
-   A customized product page with a gallery, options, and a sticky add-to-cart button.
-   A customized cart and checkout process.
-   And much more.
